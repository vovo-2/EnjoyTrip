# EnjoyTrip

## 🖥 프로젝트 소개
> **공공데이터포털의 관광지 데이터를 이용한 관광지 조회 서비스**  

- 로그인/회원가입 기능
- 시도/구군/관광지 유형에 따른 관광지 조회
- 관광지 게시판 기능
  
## 📆 개발 기간
* 23.11.19 ~ 23.11.25(약 7일)

## 🙎 팀원 및 기술스택
### FE
**허종원**
- Vue.js
- JavaScript
- sass
- library: axios, vuetify, pinia

### BE
**임소현**
- Spring Boot
- java8
- MyBatis
- MySQL
- library: lombok, swagger
